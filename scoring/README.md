# Scoring

This folder contains the scoring scripts and generated outputs for the competition.

## Inputs

- `reference_answers.csv` — the private or organizer-held truth file used for scoring
- `reference_answers.example.csv` — example truth file with synthetic values

## Outputs

- `leaderboard.csv`
- `results/round-N-scores.csv`

## Scripts

- `scripts/validate_predictions.py`
- `scripts/score_predictions.py`
