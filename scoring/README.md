# Scoring

This folder contains the reference answers, leaderboard outputs, and scoring scripts.

## Files

- `reference_answers.csv` — the organizer truth file used by the automation
- `leaderboard.csv` — the overall leaderboard generated by the workflow
- `score_log.csv` — a compact score log for each team and round
- `results/` — round-level score summaries
- `scripts/` — validation and scoring code

## Reference answers

The reference answers file must contain these columns:

```csv
challenge_round,forecast_week,hospitalizations_per_100k,r0
```

## Workflow

The GitHub Actions workflow validates prediction files, scores them against the reference answers, writes the leaderboard, and uploads the results as artifacts.
