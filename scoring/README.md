# Scoring and summaries

This folder holds the files used to summarize and score team predictions.

Common files:

- `reference_answers.example.csv` — example header for the organizer answer key
- `leaderboard.csv` — computed leaderboard
- `score_log.csv` — row-level scoring notes
- `summary.md` — optional human-readable summary

The scoring script writes the leaderboard and log files based on the reference answer key and the submission CSV files.
