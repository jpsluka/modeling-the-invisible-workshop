# Contributing

This repository supports the Modeling the Invisible workshop forecasting competition.

## If you are a participant

- Do not edit organizer data-release files.
- Submit forecasts in `predictions/<team-name>/round-N.csv`.
- Follow `docs/prediction-format.md`.

## If you are an organizer

- Update `data-release/round-N/release_info.json` for each round.
- Update `data-release/round-N/release.csv` with the released portion of the trajectory.
- Replace `scoring/reference_answers.csv` with the organizer-held truth file.
