# Data Release

This folder contains the release metadata and released data for each challenge and round.

Structure:

```text
data-release/
  challenge-01/
    challenge_info.json
    round-01/
      release_info.json
      released.csv
    round-02/
      release_info.json
      released.csv
    round-03/
      release_info.json
      released.csv
  challenge-02/
    ...
```

The release metadata file defines the forecast horizon for that round.
