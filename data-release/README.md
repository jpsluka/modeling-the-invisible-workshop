# Data Release

This folder holds the data released to participants for each challenge round.

## Structure

```text
data-release/
├── challenge-1/
│   └── released-data.csv
├── challenge-2/
│   └── released-data.csv
└── challenge-3/
    └── released-data.csv
```

Each released-data file contains the weeks available to participants for that round.

The release files are for forecasting input only. They do not replace the reference answers used for scoring.
